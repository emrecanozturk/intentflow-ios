# Roadmap

IntentFlow is release-ready as a 0.1 architecture proposal. The roadmap is intentionally focused on adoption quality.

## 0.1.x

- Core runtime
- AI manifest mode
- Generator, validator, and AI context CLI
- SwiftUI and UIKit examples
- Buildable demo app
- Migration guides
- CI, DocC, release checklist
- Multi-agent AI instruction surfaces
- Compact `intentflow ai-context` handoff command

## 0.2.0

- Manifest-to-test scaffolding
- Multi-feature parent-child flow examples
- More migration examples from VIPER, Clean Swift, and TCA
- Route interpreter helpers
- Example snapshot tests

## 0.3.0

- `intentflow validate --sources` contract/source consistency checks
- Optional Swift macros for boilerplate reduction
- DocC hosted documentation
- Example app UI tests

## Non-Goals

- Replace all MVVM usage.
- Become a large application framework.
- Hide Swift concurrency behind magic abstractions.
- Generate complete product decisions without human review.
