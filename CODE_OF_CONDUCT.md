# Code of Conduct

IntentFlow discussions should stay technical, respectful, and evidence-based.

Good contributions:

- explain the workflow pressure being solved
- include a minimal example
- discuss tradeoffs
- preserve testability
- keep AI generation reviewable

Unhelpful contributions:

- pattern tribalism
- personal attacks
- vague rewrites without behavioral motivation
- changes that make reducers impure

The project aims to be strict about architecture and generous with people.
